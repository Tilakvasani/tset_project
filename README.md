## 📋 Overview

**DocForge AI** is a full-stack AI document generation system built for the SaaS industry. It uses **LangChain + Groq (LLaMA 3.3 70B)** to generate professional documents and publishes them directly to a **Notion database** with full metadata tracking.

### ✨ Features

- 🤖 **AI-powered generation** via LangChain + Groq (LLaMA 3.3 70B Versatile)
- 📄 **12 document types** — NDA, Privacy Policy, SLA, Employment Contract, and more
- 📓 **One-click Notion publishing** with full metadata (title, type, industry, version, tags, author)
- ⚡ **Redis caching** for deduplication and fast repeated generation
- 🔁 **Rate limit backoff** — handles Notion API limits gracefully
- 🎨 **Modern dark UI** built with Streamlit
- 🐳 **Docker ready** — includes `docker-compose.yml` for full stack deployment

## 🚀 Quick Start

### Prerequisites

- Python 3.11+
- Redis (optional, for caching)
- Groq API key — [console.groq.com](https://console.groq.com)
- Notion Internal Integration token — [notion.so/my-integrations](https://www.notion.so/my-integrations)

### 1. Clone the repo

```bash
git clone https://github.com/Tilakvasani/tset_project.git
cd tset_project
```

### 2. Create virtual environment

```bash
python3.11 -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment

```bash
cp .env.example .env
```

Edit `.env`:

```env
GROQ_API_KEY=gsk_your_groq_key_here
NOTION_API_KEY=ntn_your_notion_integration_token
NOTION_DATABASE_ID=your_notion_database_id
REDIS_URL=redis://localhost:6379
APP_ENV=development
LOG_LEVEL=INFO
```

### 5. Run the backend

```bash
uvicorn backend.main:app --reload
```

### 6. Run the frontend (new terminal)

```bash
streamlit run ui/streamlit_app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/generate` | Generate a new document |
| `POST` | `/api/publish` | Publish document to Notion |
| `GET`  | `/api/library` | Fetch all docs from Notion |

### Example — Generate Document

```bash
curl -X POST "http://localhost:8000/api/generate" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Master Service Agreement",
    "industry": "SaaS",
    "doc_type": "NDA",
    "description": "B2B NDA for enterprise SaaS clients",
    "tags": ["legal", "b2b"],
    "created_by": "admin"
  }'
```

## 🗃️ Notion Database Schema

Every generated document is stored in Notion with:

| Field | Type | Description |
|-------|------|-------------|
| Title | Title | Document name |
| Doc Type | Select | One of 12 document types |
| Industry | Select | SaaS (locked) |
| Version | Number | Document version |
| Tags | Multi-select | Categorization tags |
| Created By | Text | Author name |
| Created At | Created time | Auto timestamp |
| Status | Select | Draft / Generated / Reviewed / Archived |
| Word Count | Number | Auto-calculated |
| Source Template ID | Text | Template reference |
| Doc ID | Auto ID | Auto-incremented `DOC-N` |
